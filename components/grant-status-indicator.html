Guidelines for use

Purpose:

Displays the status of a grant or program. Either a green tick icon and ‘Open’, or a red no-entry icon and ‘closed’.

When to use this component:

This is added automatically whenever the status of the program is defined on the program page meta content.

Layouts

Page layouts
