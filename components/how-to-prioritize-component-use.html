Priority tasks 

When used in a content page, the pathway component type is selected based on the priority of the link or task it provides. 

Priority 1 = High value hero of the page – link/task that is important and the user must do. E.g registration, applying 

Priority 2 = Medium value – link/task that is important and worth highlighting, but is optional to the user. E.g. reading more information about a topic 

Priority 3 = Low value optional – link/task that optional that the user is not required to do. E.g. Before applying/registering, consider talking to a legal professional or business adviser.
