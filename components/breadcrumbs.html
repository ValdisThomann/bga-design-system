Guidelines for use

Purpose:

The breadcrumb menu is a set of navigation links that show the user where they are in the site structure. It displays all links in the hierarchy between the current page and the home page.

When to use this component:

  • Used across every page of the website to ensure a consistent user experience.

Layouts

Page layouts

Breadcrumbs are used full width and 2/3 width pages. They are on every page except the home page.

Component layout options

Full width layout

Desktop, Tablet & Mobile: One component in a row.
