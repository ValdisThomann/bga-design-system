Guidelines for use.

Purpose

The progress indicator is used to visually show the user how far they have progressed through a process. Progress indicators are not used for navigation.

When to use this component

Un-named steps progress indicator

This progress indicator is the default indicator and can be used to show progress through a complete process.

Named steps progress indicator

This progress indicator can be used to show progress of sub-steps within a larger process. Naming each step makes it clear to the user that the indicator is for that particular step, rather than the whole process.
