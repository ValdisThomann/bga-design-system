Guidelines for use.

Purpose:

The header includes the site logo, main navigation and search. It is designed to make the site easy to nagivate and establishes the site brand.

When to use this component:

  • Used across every page of the website to ensure a consistent user experience.

Layouts

Page layouts

The global header is used on full width and 2/3 width pages.

Component layout options

Full width layout

Desktop, Tablet & Mobile: One component per page. The global header background colour is wider than the body content and extends the full width of the screen.
