Notes

The anchor menu is uses persistent behaviour - it stays in view as the user scrolls through the page content. At mobile size the anchor menu is not persistent.

This menu always has a heading titled "On this page".

Guidelines for use

Purpose:

Used to define sections of a page and add a menu item to the left-hand side navigation – on the same page.

When to use this component:

  • Link names usually match the heading on page (this can be shortened if needed)
  • Headings are usually H2

Layouts

Page layouts

The anchor menu is used on 2/3rd width pages.

Component layout options

Single column layout

Desktop & tablet: One column, to the left of the body content.

Mobile: One column that spans the full width of the screen.
