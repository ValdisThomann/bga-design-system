Guidelines for use

Purpose: 

Feature image pathways are used to promote content (often on a temporary basis). They provide a high profile pathway (with an image) to another page on the website.   

When to use this component: 

  • Feature image pathways are most commonly used to promote particular news articles on the home page. 

Content rules: 

  • Display description. 
  • Display content type label. 
  • No alt text for pathway image.

Layouts

Page layouts

Feature image pathways can be used in full width pages.

Component layout options

Full width layout

Desktop, Tablet & Mobile: One component on a page that stretches the full width of the content area.
