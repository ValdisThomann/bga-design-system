Guidelines for use

Purpose:

Used to display advisor details in a pre-defined layout.

When to use this component:

  • On pages that require a list of advisor profiles, that include their contact details and a photo.

Layouts

Page layouts

Advisor profiles are used in two-third width pages.

Full width layout
