Guidelines for use

Purpose: 

The image pathway list component is used to provide pathways to a list of pages on the site, using tiles with a supporting image. It has an optional button below the image pathways. 

When to use this component: 

  • Image pathway components are commonly used on the home, news and customer story index pages. They provide navigation tiles (each with an image) to a list of similar page types (such as news articles or customer stories).  
  • Image pathways may also be used on content pages for priority 2 tasks (for example, to list customer stories for a particular grant). 

Content rules: 

  • Display description for each item. 
  • Do not display content type label, except on news index page. 
  • No alt text for pathway images. 

Layouts

Page layouts

Image pathway list components can be used in full width and 2/3rd width pages.

Component layout options

Equal width layout (on full width pages)

Desktop: Three equal width components in a row.

Tablet: Two equal width components in a row.

Mobile: One equal component in a row.

Equal width layout (on 2/3rd width pages)

Desktop & tablet: Two equal width components in a row.

Mobile: One equal component in a row.
