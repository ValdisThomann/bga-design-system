Standard pathway

The standard pathway is an in-page link to another page on the site.

Variations

The default heading is H3 but it can also be H4. Refer to the typography guidelines.

Guidelines for use

Purpose: 

Generally used as a default option within index pages to link to a list of pages on the website.  

When to use this component: 

  • Used primarily for category or topic index pages. 
  • May also be used to show page hierarchy when a level that is lower than hero and higher than light pathways is required. 
  • Standard pathways may be used on content pages when linking to 3 or more internal links under the same heading. 
  • When used on content pages, the links should be for priority level 2 tasks or content.

Layouts

Page layouts

Standard pathway components are used in full width pages.

Component layout options

Equal widths layout

Desktop: Three equal width components in a row.

Tablet: Two equal width components in a row.

Mobile: One component in a row.
