Guidelines for use.

Purpose:

The cross-site header provides a link to the main homepage of business.gov.au from sub-sites, or tools. It can also provide additional sister site links if required.

When to use this component:

  • Used across every page of a busniess.gov.au sister site or tool to ensure a consistent user experience.

Layouts

Page layouts

Present on all pages of a business.gov.au sister site.

Component layout options

Full width layout

Desktop, Tablet & Mobile: One component per page. The header background colour is wider than the body content and extends the full width of the screen.
