Guidelines for use.

Purpose:

Provides navigation, social media and contact options at the bottom of site pages.

When to use this component:

  • Used across every page of the website to ensure a consistent user experience.

Layouts

Page layouts

The global footer is used on full width and 2/3 width pages.

Component layout options

Full width layout

Desktop, Tablet & Mobile: One component per page. The global footer background colour is wider than the body content and extends the full width of the screen.
