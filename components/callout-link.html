Guidelines for use

Purpose:

The call out link component highlights a single link in a call out box, with an icon to show the destination type (e.g. email subscription or web page).

When to use this component:

  • Call out links are generally used to promote subscription links for newsletters, updates and alerts.
  • They may be used on both index and content pages.

Layouts

Page layouts

Call out components can be used in full width and 2/3rd width pages.

Note: In full width layouts this component is centered on the page with a maximum width of 620px.
